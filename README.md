# NF-Me

NF-Me is a tool used to prove you are you.

### Quickstart:

To verify

```sh
NF-Me -v
```

To create a key

```sh
NF-Me -c
```

To sign a message

```sh
NF-Me -s
```

Arguments:

```
-h --help
-c --create public key
-s --sign message
-v --verify message
-p --public key file name
-f --signature file name
```
